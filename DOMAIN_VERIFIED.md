### Common Issues:

**1. "Site Not Found" or "Invalid Domain"**
- Wait for DNS propagation (can take up to 48 hours, usually 15-60 min)
- Verify DNS records are correct at your registrar

**2. SSL Certificate Not Active**
- Vercel automatically provisions SSL
- Usually active within 5-10 minutes of domain verification
- Check domain status in Vercel dashboard

**3. Site Shows Default Vercel Page**
- Make sure your project is deployed
- Check "Deployments" tab in Vercel
- Ensure latest deployment is successful

## Troubleshooting

### Domain Shows "Proxy Detected" but Site Not Loading:

1. **Check DNS Propagation:**
   ```bash
   dig graphclic.com A
   # Should return: 76.76.21.21
   ```

2. **Check Vercel Deployment:**
   - Go to "Deployments" tab
   - Ensure latest deployment is successful
   - Check for build errors

3. **Clear Browser Cache:**
   - Hard refresh: Cmd+Shift+R (Mac) or Ctrl+Shift+R (Windows)
   - Or try incognito/private window

4. **Wait for Propagation:**
   - DNS can take up to 48 hours (usually 15-60 minutes)
   - SSL certificate takes 5-10 minutes after DNS propagates
